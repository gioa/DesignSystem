# Other Variables

**Disable opacity**

```css
const disabledOpacity = {
    small: 0.34,
    regular: 0.45, 
}
```

##### Z-index

```css
const z-index = {
    hide: -1,
    default: 0,
    small: 1,
    modal: 500,
    dropmenu: 999,
    tooltip: 1000,
    loadingIndicator: 1001,
}
```

```css

};
```
